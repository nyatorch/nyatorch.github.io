# KiCad
KiCad (/ˈkiːˌkæd/ KEE-kad) 是用于 PCB 设计的[自由软件]（https://www.gnu.org/philosophy/free-sw.zh-cn.html）
作为开源工具，在国外的开源硬件项目中使用广泛  
之前看有 adi 大佬拿来画fpga和ddr的，我的水平估计是找不到不够用的地方了
